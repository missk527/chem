Notes (Key points rewritten from the speech)

Mendeleev and the purpose of the table
  • Dmitri Mendeleev (Russian chemist) devised the periodic table in the 19th century.
  • It organised known elements and even predicted undiscovered elements.
  • All substances are made from different combinations of elements in the table.
How the table is read and what each box shows
  • Read like a book: left → right, then top → bottom.
  • Elements are arranged by increasing atomic number (number of protons).
  • Typical box shows: symbol (e.g., Na), atomic number (bottom‑left), and mass number (top‑left).
  • Atomic number = number of protons; mass number = protons + neutrons.
Patterns → periods and groups
  • Repeating patterns of chemical properties led to starting a new row called a period.
  • Elements in the same vertical column (group) have similar chemical properties.
  • Groups are numbered from 1 to 7, with the far right called group 0 (noble gases).
  • The block between groups 2 and 3 are the transition metals (often unnumbered).
Outer electrons and reactivity trends
  • Elements in the same group have the same number of outer‑shell (valence) electrons.
  • For main‑group elements, the group number = number of outer electrons.
  • Group 1 (alkali metals): 1 outer electron; similar properties (e.g., react vigorously with water); reactivity increases down the group.
  • Group 7 (halogens): 7 outer electrons; reactivity decreases down the group.
  • Group 0 (noble gases): full outer shells → very unreactive.
Metals vs non‑metals & layout variations
  • A zig‑zag line (Al → Po) often separates metals (left) from non‑metals (right). There are more metals than non‑metals.
  • Transition metals sit in the centre block.
  • Different periodic tables may position numbers differently, but atomic number is the smaller number in an element box.
